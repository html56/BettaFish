## v1.2.0 - 布局比例优化 (2026-01-11)

### 主要更新
- **调整左右面板布局比例**: 将左侧"分析结果"面板和右侧"系统控制台"面板的宽度比例从 1:1 调整为 2.5:1 (约71%:29%)
- **提升内容展示效果**: 左侧面板占据更多空间,更适合展示分析结果和报告内容
- **优化空间利用**: 右侧控制台面板保持紧凑,节省屏幕空间的同时保持良好的可读性

### 技术改进
- 在 HTML `<style>` 标签中添加 `!important` 规则,强制设置布局比例
- 修改 CSS 文件中主样式和媒体查询中的 flex 值,确保响应式设计的一致性
- 优化配置弹窗关闭逻辑,添加系统启动状态自动检测
- 更新 CSS 版本号为 v3.1,解决浏览器缓存问题
- 修复 Flask 模板缓存导致的样式不生效问题

### 文件变更
- `templates/index.html`: 添加布局比例强制规则,优化弹窗关闭逻辑
- `static/modern_style_new.css`: 更新主样式和媒体查询中的 flex 值

### 视觉效果
- 左侧面板宽度: 约 71% (flex: 2.5)
- 右侧面板宽度: 约 29% (flex: 1)
- 整体布局更加符合内容展示需求,用户体验得到显著提升
