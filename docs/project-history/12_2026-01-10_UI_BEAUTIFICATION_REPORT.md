## 设计方案

### 核心设计理念

采用**深色科技主题**配合**毛玻璃效果（Glassmorphism）**，通过渐变色彩、发光效果和流畅动画，打造具有未来感的专业级界面。

### 视觉设计特点

#### 1. 色彩系统

**主色调**：深蓝紫色系
- 背景色：#0f0f23（深空蓝）
- 主渐变：#667eea → #764ba2（紫蓝渐变）
- 辅助渐变：#f093fb → #f5576c（粉红渐变）

**强调色**：
- 青色：#00d4ff（用于高亮和发光效果）
- 紫色：#667eea（用于按钮和边框）
- 粉色：#f5576c（用于警告和特殊操作）

**状态色**：
- 成功：#10b981（绿色）
- 警告：#fbbf24（黄色）
- 错误：#ef4444（红色）

**文字色**：
- 主文字：#ffffff（纯白）
- 次要文字：#a0aec0（灰蓝色）

#### 2. 毛玻璃效果

所有主要界面元素都采用半透明背景配合 `backdrop-filter: blur()` 实现毛玻璃质感：

- 搜索区域：rgba(15, 15, 35, 0.8) + blur(20px)
- 按钮卡片：rgba(255, 255, 255, 0.1) + blur(10px)
- 配置模态框：rgba(15, 15, 35, 0.95) + blur(20px)
- 控制台背景：rgba(0, 0, 0, 0.4)

#### 3. 渐变与发光

**渐变文字**：
- 标题使用渐变色填充（background-clip: text）
- 配合 text-shadow 增强视觉冲击力

**发光效果**：
- 按钮悬停：box-shadow 配合半透明颜色
- 输入框聚焦：0 0 20px rgba(0, 212, 255, 0.2)
- 边框发光：使用 rgba 颜色实现柔和发光

#### 4. 圆角设计

- 按钮：12px 圆角
- 输入框：10px 圆角
- 模态框：20px 圆角
- 状态指示器：50%（圆形）

### 交互设计特点

#### 1. 动画效果

**过渡动画**：
- 统一使用 0.3s 过渡时间
- 采用 cubic-bezier(0.4, 0, 0.2, 1) 缓动函数
- 确保流畅自然的交互体验

**关键帧动画**：
- 脉冲动画（pulse）：用于状态指示器
- 淡入动画（fadeIn）：用于控制台输出
- 滑入动画（slideUp）：用于模态框出现

#### 2. 悬停效果

**按钮悬停**：
- 上移 2px（transform: translateY(-2px)）
- 产生发光阴影
- 背景渐变反转

**输入框聚焦**：
- 边框颜色变为青色
- 产生发光效果
- 背景透明度增加

**标签页切换**：
- 底部出现渐变下划线
- 宽度从 0 过渡到 80%
- 背景色轻微变化

#### 3. 状态反馈

**状态指示器**：
- 使用彩色圆点（8x8px）
- 配合脉冲动画
- 发光效果（box-shadow: 0 0 10px currentColor）

**加载状态**：
- 控制台文字逐行淡入
- 青色文字配合深色背景
- 等宽字体呈现终端风格

### 排版设计

#### 1. 字体系统

**主字体**：Inter（Google Fonts）
- Fallback：-apple-system, BlinkMacSystemFont, 'Segoe UI'
- 权重：300, 400, 500, 600, 700

**代码字体**：SF Mono, Monaco, Courier New
- 用于控制台输出
- 等宽字体确保对齐

#### 2. 字号体系

- 标题：32px（search-title）
- 模态框标题：20px
- 正文：14-15px
- 小字：12-13px

#### 3. 间距系统

- 大间距：30px（section padding）
- 中间距：20px（field margin）
- 小间距：12px（gap）
- 微间距：8px（内部间距）

### CSS 技术特性

#### 1. CSS 变量（Custom Properties）

```css
:root {
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --dark-bg: #0f0f23;
    --text-primary: #ffffff;
    /* ... 更多变量 */
}
```

**优势**：
- 便于主题管理和切换
- 提高代码可维护性
- 支持动态修改

#### 2. Flexbox 布局

全面使用 Flexbox 实现响应式布局：
- 搜索区域：flex-direction: row
- 主内容：flex-direction: row（桌面）/ column（移动）
- 模态框：flex-direction: column

#### 3. 伪元素技术

**::before 伪元素**：
- 搜索区域顶部渐变线
- 按钮悬停渐变背景
- 标签页底部指示器

**::after 伪元素**：
- 锁定按钮的锁图标占位
- 其他装饰性元素

#### 4. 关键帧动画

```css
@keyframes pulse {
    0%, 100% { opacity: 1; }
    50% { opacity: 0.5; }
}

@keyframes fadeIn {
    to { opacity: 1; }
}

@keyframes slideUp {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
```

#### 5. 响应式设计

```css
@media (max-width: 1024px) {
    .main-content {
        flex-direction: column;
    }
}
```
